## Overview

Recon is a full-stack web application designed to provide detailed and actionable analytics on UFC fighters. The platform delivers fighter performance metrics,
historical trends, and percentile-based rankings across major statistical categories. Users can explore fighter profiles,
compare skills across divisions and time periods, and gain unforeseen insights for match-up analysis. Recon is designed for analysts, fans, and enthusiasts who want a deeper understanding of
fighter performance beyond basic stats.

## Features

- Fighter Profiles: View detailed stats, historical performance trends, and key metrics for each UFC fighter.
- Search & Filtering: Quickly search for fighters then filter by division and time period to see metrics
- Percentile Rankings: Compare fighters across divisions using percentile-based rankings for intuitive benchmarking.
- Performance Visualization: Interactive charts and visualizations to highlight trends, tendencies, and match-up insights.
- Real-Time Analytics: RESTful API serves structured fight data, enabling instant access to analytics tools and web applications.
- Full-Stack Architecture: React/Next.js frontend with Flask backend and PostgreSQL database ensures scalable and responsive performance.
